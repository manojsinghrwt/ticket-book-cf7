CF7 add-on to book ticket

== Description ==

This is simple ticket booking add-on for Contact Form 7 with default 100 tickets.

This plugin add CF7 Form tag [ticket_book_cf7] so it easy to use with few click.


A few notes:

*   Work with Contact Form 7
*   Intall Contact Form 7 Latest version
*   "Requires at least wp 4.8" is the lowest version that the plugin will work on
*   "Tested up to wp 5.3.2 version" is the highest version that you've *successfully used to test the plugin*. Note that it might work on
higher versions... this is just the highest one you've verified.

== Installation ==

To install the plugin and get it working follow below step.

1. Download the zip file and extract it.
2. Upload `ticket-book-cf7` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Now plugin is ready to use check CF7 and [Ticket Book] tag to use
